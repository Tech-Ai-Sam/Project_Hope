# 🌟 Project HOPE

> A safe, non-judgmental, and deeply comforting digital space to map your inner climate and speak with an empathetic AI companion.

Project HOPE is a premium, immersive dark-themed mobile app designed to help users track their mental wellness and process thoughts safely.

## 🛠️ Tech Stack & Architecture
* **Frontend:** Flutter & Dart (Clean Architecture, Flat Self-Contained Views)
* **Backend Integration:** Groq API (Compatibility Layer (`/v1`) via `dart_openai`)
* **LLM Core:** `llama-3.3-70b-versatile` with highly tuned prompt-engineered empathy constraints
* **Data Persistence:** `shared_preferences` (JSON-serialized chat caching)
